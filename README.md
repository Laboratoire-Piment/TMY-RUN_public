# TMY-RUN_public
Ce dépôt met à disposition des années météorologiques types (TMY) générées pour plusieurs stations météorologiques de l’île de La Réunion, à partir de données issues de stations de mesure au sol.

Les stations disponibles à la dernière mise à jour du 16/12/2025 sont :
- GILLOT-AEROPORT
- PIERREFONDS-AEROPORT
- BELLEVUE_BRAS-PANON
- COLIMACONS
- LE_BARIL
- LE_PORT
- PITON-MAIDO
- PLAINE_DES_CAFRES
- PONT-MATHURIN
- SAINT-BENOIT
